**L**iwei's **S**cheme **I**nterpreter.

## Build from scratch

Use LSI for painting!

```
git submodule update --init
mkdir build && cd build && cmake .. -DCMAKE_BUILD_TYPE=Release
make LSI_CLI
../build/Interface/CLI/src/LSI_CLI -o res.bmp cli_test/CLITest.scm -p stdlib
```
